# 🛒 Web Sales Management System

A web-based sales management system built using HTML, CSS, and JavaScript with REST API integration for managing products efficiently.

## 📌 Overview

This project provides a simple and interactive interface for managing product inventory and sales information. Users can create, update, delete, search, and manage products through a responsive web application connected to a REST API.

## ✨ Features

* Add new products
* Update existing products
* Delete individual products
* Delete all products
* Search products by name
* Dynamic total price calculation
* Bulk product creation using quantity input
* REST API integration using Fetch API
* Real-time UI updates without page reload

## 🧮 Product Pricing Formula

```text
Total Price = Price + Taxes + Ads - Discount
```
